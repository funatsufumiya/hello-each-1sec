# hello-each-1sec

just print hello each sec (ex: for testing use)

## Usage

```bash
$ hello-each-1sec -h
    -h: show help
    -e: print to stderr
    -t [sec]: specify exit sec (default: 0 / means no exit)
```

## Build and Install

```bash
$ cargo build # build
$ cargo install --path . # install
```
